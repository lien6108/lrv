# 地圖與房間

飛船版圖由多個房間以走廊相連構成，不同房間提供不同行動，是整個遊戲的活動場所。

---

## 版圖基本概念

**房間**：版圖磁磚上的區格，角色可停留並執行行動。
**走廊**：連接兩個房間的通道，允許角色自由移動。
**門**：部分走廊設有門，移動時需額外花費 1 行動點，或使用相關能力通過。
**檢修通道**：每個房間側面的細小通道，僅供噪聲傳播和部分特定規則使用，**角色與入侵者不可移動至此**。

---

## 特殊房間一覽

| 房間 | 主要行動 |
| --- | --- |
| **休眠室** | 進入休眠（完成個人目標的途徑之一） |
| **駕駛艙** | 設定航行座標 |
| **引擎室（A、B）** | 修復引擎損壞 |
| **武器庫** | 取得武器 |
| **醫療室** | 包紮 / 治愈嚴重創傷；幼蟲移除 |
| **實驗室** | 掃描污染卡牌 |
| **減壓艙門控制** | 開啟 / 關閉逃生艙 |
| **餐廳 / 浴室** | 移除污染卡牌 |
| **計算機房間** | 執行各類系統指令（依版圖說明） |

---

## 探索房間

**未探索的房間**以正面朝下放置，翻開條件：

1. 角色執行移動行動**進入**未探索房間
2. 翻開磁磚，如有探索標記一同揭示
3. 依探索標記圖示結算（放置火焰、故障、幼蟲，或無事發生）

> ⚠️ 部分探索標記有即時效果，翻開後立刻結算再繼續行動。

---

## 噪聲傳播路徑

當需要「在所有連接通道放噪聲標誌」時：

- 計算目標房間直接連接的**走廊**與**檢修通道**的數量
- 在每條通道的目標房間端放置 1 個噪聲標誌
- 若通道已有噪聲標誌，觸發**遭遇**（見遭遇異形）

---

## 房間行動限制

- 同一回合每個房間的行動**只能使用一次**（除非有卡牌說明可重複）
- 有**故障標誌**的房間行動無法執行
- 有**火焰標誌**的房間行動仍可執行，但回合結束時角色受 1 輕微創傷
