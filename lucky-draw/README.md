# 抽獎系統 (Lucky Draw System)

這是一個具備前後端分離架構的網頁版抽獎系統。
前端支援多人即時參與，並能依據指定順序進行抽獎。
後端提供加密的系統管理員介面，可用於管理獎品、設定抽獎順序、匯出 CSV 抽獎結果，以及分配特定獎品池給特定人員。

## 目錄結構

- `client/`: 前端 React (Vite) 專案
- `server/`: 後端 Node.js (Express) 與 SQLite 資料庫
- `run.bat`: 快速啟動前後端的批次檔

## 環境需求

- Node.js (建議 v16 或以上版本)
- npm

## 安裝與執行

### 1. 安裝套件

請分別進入 `client` 與 `server` 目錄安裝所需的 Node.js 套件：

```bash
# 安裝前端套件
cd client
npm install

# 回到根目錄並安裝後端套件
cd ../server
npm install
```

### 2. 啟動系統

您可以直接在根目錄點選或執行 `run.bat` 來同時啟動前端與後端伺服器（此批次檔會自動開啟兩個終端機視窗）：

```cmd
run.bat
```

或者，您也可以手動分別啟動：
- **前端 (Client)**: 進入 `client` 資料夾並執行 `npm run dev`
- **後端 (Server)**: 進入 `server` 資料夾並執行 `node index.js`

## 部署建議

本系統在設計上支援前後端分離部署，建議方式如下：
- **前端**: 建議部署至靜態網頁代管服務，例如 GitHub Pages, Vercel 或 Netlify。
- **後端**: 建議部署至支援 Node.js 的雲端平台，例如 Render 或 Railway。

## 特色功能

- **多人即時互動**: 支援多位使用者同時操作與觀看抽獎過程。
- **管理員後台**: 具備密碼保護，確保活動設定與資料安全。
- **獎項管理與指定**: 可於後台新增、修改、刪除獎品，並能設定特定獎項供特定抽獎人抽取。
- **資料匯出**: 支援將抽獎結果一鍵匯出為 CSV 格式，方便後續作業與記錄。
- **響應式設計**: 介面與表格皆支援自適應設計 (RWD)，相容各種裝置瀏覽。
- **全繁體中文介面**: 提供在地化、友善的使用者體驗。
